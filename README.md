# Dự án: Xây dựng mô hình máy học để dự báo giá tiền điện tử
## Mô tả dự án
Dự án này xây dựng một mô hình học sâu để phân tích và dự báo xu hướng giá của các loại tiền điện tử như Bitcoin, Ethereum,... Kết quả được triển khai trên một giao diện web để người dùng dễ dàng theo dõi.
## Chức năng chính
- **Phân tích dữ liệu:** Xử lý và trực quan hóa dữ liệu lịch sử giá tiền điện tử.
- **Mô hình dự đoán:** Ứng dụng thuật toán học sâu Long Short-Term Mamory (LSTM) đã được huấn luyện (tích hợp trong file `model.keras`) để phân tích và đưa ra các dự báo trong tương lai.
- **Giao diện web:** Tương tác trực quan thông qua ứng dụng web được xây dựng bằng **Flask**.
- **Dễ dàng triển khai:** Cấu trúc code gọn nhẹ, dễ dàng chạy trên môi trường local.
